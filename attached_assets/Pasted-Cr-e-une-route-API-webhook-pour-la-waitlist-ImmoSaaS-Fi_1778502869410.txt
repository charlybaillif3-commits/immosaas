Crée une route API webhook pour la waitlist ImmoSaaS.

Fichier à créer : app/api/waitlist/route.ts

Fonctionnement :
1. Reçoit un POST webhook depuis Tally avec les données du formulaire
2. Extrait email et nom_agence depuis le payload Tally
3. Envoie un email de confirmation via Resend à l'email de l'inscrit
4. Retourne { success: true }

Format webhook Tally :
{
  "data": {
    "fields": [
      { "label": "Votre email professionnel", "value": "email@agence.fr" },
      { "label": "Nom de votre agence", "value": "Agence Dupont" }
    ]
  }
}

- Subject : "Votre accès beta ImmoSaaS est réservé ✓"
